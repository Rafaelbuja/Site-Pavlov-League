False Allies é penalizado - 12/01/2025

Após a revisão de uma jogada no Replay do PavlovTV foi observado o uso indevido de um Glitch no mapa Dust 2 pelo jogador Gulix do time False Allies. Após conversas com os capitães e organização foi aplicada uma punição descrita nas regras da Liga onde False Allies teve que entregar seu round ganho para o GIGANTES deixando assim o placar final do jogo em 9x2 para False Allies. O jogador Gulix se pronunciou dizendo que não foi intencional, mas devido a vantagem obtida no momento a penalização foi aplicada e o jogador levou um "Cartão Amarelo".
